# NAGARA PRO トップページ全面リデザイン

出張洗車・コーティングのマッチングサービス「NAGARA PRO」のトップページを、コンバージョン率が高くユーザーフレンドリーなモダンUIに書き換える。美容室予約サイトのような「エリアや条件から直感的に職人を探せる体験」を目指す。

## 設計方針

- **既存のlist/profile/bookingビュー**はそのまま維持し、**homeビュー（トップページ）のみ**を全面改修
- ユーザー要件の全7セクションを上から順に配置
- スマートフォンファーストのレスポンシブデザイン
- 既存のデザインシステム（カラー、フォント、アニメーション）を活用しつつ、新セクション用のスタイルを追加

## 変更の概要（全7セクション）

| # | セクション | 内容 |
|---|---|---|
| 1 | ヘッダー＆ナビ | ロゴ + ログイン/会員登録 + マイカー登録ボタン |
| 2 | ファーストビュー | 背景画像 + キャッチコピー + エリア検索UI（都道府県・市区町村） |
| 3 | サービスの特徴 | アイコン付き3メリット（高品質施工/明朗会計/選べる施工場所） |
| 4 | メニュー＆料金 | 梅竹松特上の4コース比較カード（松をハイライト） |
| 5 | おすすめ職人一覧 | 横スクロールカード + 出張OK/持ち込みOKバッジ |
| 6 | ご利用の流れ | 4ステップのフロー |
| 7 | CTA + フッター | 「傷なんて磨けばいい。」+ 職人を探すボタン |
